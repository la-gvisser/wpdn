Welcome to Engine Analysis Module  Wizard for Visual Studio

 

 Select options:   Description: 
    Plugin Messages    Send and respond to private plugin messages.
    Engine and Context Tab    Create tabs in the Engine and Capture windows.
    HTML Dialogs (no Console Context)    Plugin only installs on OmniEngine. (No Summarize Packet.)
       Cef Handler      HTML Handler     Render webpages with Chrome or Internet Explorer.
    Include Example Code    //#define IMPLEMENT_CODEEXAMPLES in StdAfx.h to remove.

 Remote Processing:   Description: 
    Reset Capture    Plugin is called when the packet count has rolled-over.
    Start, Stop and Reset Capture    Plugin is called when a capture is started, stopped or reset.
    Filter Packets    Plugin is called to filter packets. Allows use in Advanced Filters.
    Process Packets    Plugin is called to process packets.
    Time Notification    Plugin is called once-per-second during capture. Simulated for files.
    Summary Statistics    Plugin is called to provide summary statistics.
    Send and Receive Notifications    Plugin receives and can send notifications.

 Console Processing:   Description: 
    Summarize Packet    Plugin can supply a packet summary.
