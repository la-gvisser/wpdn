CLayerIP Class

The CLayerIP class provides access to IP layer properties.

class CLayerIP

:   public CLayer

Constructors
CLayerIP Constructs a CLayerIP object.

Public Member Functions
GetDestination Returns the destination IP Address.
GetHeader Returns the IP Header as an object.
GetHeaderLength Returns the number of bytes in the IP Header layer.
GetIp4Header Returns the IPv4 Header as an object. If the layer is not IPv4 then the object will be invalid.
GetIp6Header Returns the IPv6 Header as an object. If the layer is not IPv6 then the object will be invalid.
GetPayloadLength Returns the number of bytes in the layer's payload.
GetPayloadProtocol Returns the protocol type of the payload.
GetSource Returns the source IP address.
GetVersion Returns the object's IP version (IP version 4 or 6).
IsFragment Returns true if the payload is a fragment.
IsNotValid Returns true if this object is not valid.
IsValid Returns true if this object is valid.

CLayerIP::IsFragment

Is the packet a fragment.

bool IsFragment() const;

Return Value

True if the packet is a fragment.

Remarks

This function differs from Peek's idetification of a fragment in that it includes the first packet. Fragmentation is not detected in IP V6 packets.
