CLayer Class

CLayer is the base class for other more specific classes. A layer contains a header and a payload. This class provides standard public methods for the layer type and accessing the header and payload data.

GetDataRef Returns a CByteVectRef of the payload of the layer.

GetHeaderLength Returns the number of bytes in the header of the layer.
GetHeaderRef Returns a CByteVectRef of the header of the layer.
GetLength Returns the total length, header and payload, of the layer.
GetOffset Returns the offset of the first byte of the header of the layer within the packet.
GetPacket Returns a CPacket of the packet that this layer is a part of.
