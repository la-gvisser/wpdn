Engine Plugin Wizard Installation Guide


Installation

  1. Download the Engine Plugin Wizard Installer (EngineWizardInstaller11.exe) from MyPeek.Savvius.com.
  2. Run the installer and click the Setup button to begin the installation process.
  3. Click the Next button to continue.
  4. The supported versions of Visual Studio will be displayed. Check the checkbox next to each version of Visual Studio you would like the Engine Plugin Wizard to be installed. Then click the Install button to install the Engine Plugin Wizard.
  5. The installer will display the resulting status of the installation. Click the Finish button to exit the installer.

Uninstall

  1. Run the installer and click the Setup button to begin the installation process.
  2. Click the Next button to continue.
  3. The supported versions of Visual Studio will be displayed. The Uninstall button will be enabled if the Engine Plugin Wizard is installed in the corrisponding version of Visual Studio. Check the checkbox next to each version of Visual Studio you would like the Engine Plugin Wizard to be removed from. Click the Uninstall button to remove the Engine Plugin Wizard from the selected versions of Visual Studio.
  4. The installer will display the resulting status of the uninstall. Click the Finish button to exit the installer.
