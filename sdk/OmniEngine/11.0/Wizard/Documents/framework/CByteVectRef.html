CByteVectRef Class

The CByteVectRef class contains a pointer to an array of bytes and a count of how many bytes are in the buffer. The inMin argument to the GetData member helps ensure that the array contains at least the number of bytes needed.

CByteVectRef::operator CString()

Returns a copy of the buffer as a CString.

operator CString() const;

Remarks

The CString operator copies the entire contents of the buffer into a CStringA object and returns a CString object created from the CStringA object. The length of the string will be the number of bytes in the data or the offset of the first null character, whichever is smaller.

CByteVectRef::GetData

Returns a pointer to the data.

const UInt8* GetData(
     UInt32 inMin
);

Parameters

inMin
The minimum number of bytes that must be in the buffer.

Remarks

If there are at least inMin bytes in the buffer, then the function returns a pointer to the buffer. Otherwise it returns NULL.

CByteVectRef::IsEmpty

CByteVectRef::Test

Verifies the validity of the object.

void Test(
  UInt32 inMax = 0xFFFFFFFF
);

Parameters

inMax
The maximum number of bytes that the object should reference.

Remarks
