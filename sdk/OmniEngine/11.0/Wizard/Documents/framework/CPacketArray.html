CPacketArray Class

The CPacketArray is an unsorted collection of CPackets.

class CPacketArray
   : public CAtlArray

Constructors
CPacketArray  

Operators
operator = Returns the object as an operation.

Public Member Functions
Add Copies the packet to the end of the collection.
Append Copies the packets from the other collection to the end of this collection.
Copy Copies the other collection to this collection.
Detach Detaches all of the attached packets in the collection.
GetType Returns the type of collection: mixed, attached or detached.
Insert Inserts a copy of the packet to the index.
IsAttached Returns true if all of the packets in the collection are attached.
IsDetached Returns true if all of the packets in the collection are detached.
IsMixed Returns true if the collection contains both attached and detached packets.
Load Adds all of the packets from the file.
ReplacePacket Replaces the packet at the index with a copy of the packet.
