CPeekEngineContext::OnFilter

The framework calls this method to filter a packet.

  CPacket inPacket
  UInt16* pcbBytesToAccept
  CGlobalId& outFilterId
);

Parameters

inPacket
The packet to filter.
pcbBytesToAccept
The number of bytes of the packet to accept. Set to zero to accept the entire packet.
outFilterId
The GUID of the filter. Ignored.

Return Value

Return PEEK_PLUGIN_SUCCESS to accept the packet. Return PEEK_PLUGIN_FAILURE to reject the packet.
