Using the Engine Plugin Wizard


Create a New Plugin

  1. From Visual Studio select the File | New | Project... menu.
  2. Select from the Installed | Templates | Visual C++ from the sidebar.
  3. Select Engine Plugin 11 from the Templates list.
  4. Set the Name, Location, Solution and Solution Name. Click the OK button to continue.
  5. Set the checkboxes for the new project. The Select Common button checks the most commonly used options, but switch to the HTML Handler option.
  6. Click Finish to create the new plugin.
